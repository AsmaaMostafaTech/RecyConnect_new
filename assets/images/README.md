# Images Directory

This directory contains image assets for the RecyConnect application.

## Recommended Images

### Favicon
- `favicon.ico` - 16x16, 32x32, 48x48 sizes
- Format: ICO file with multiple sizes

### Apple Touch Icon
- `apple-touch-icon.png` - 180x180 pixels
- Format: PNG

### Open Graph Image
- `og-image.jpg` - 1200x630 pixels
- Format: JPG or PNG
- Used for social media sharing

### Logo
- `logo.png` - High resolution logo
- Recommended size: 512x512 pixels
- Transparent background preferred

## Image Optimization Tips

1. **Compress images** before adding to this folder
2. **Use appropriate formats**: 
   - PNG for logos and icons with transparency
   - JPG for photographs
   - WebP for modern browsers (optional)
3. **Maintain consistent styling** across all images
4. **Test at different screen sizes** for responsiveness

## Adding New Images

1. Place images in the appropriate subfolder
2. Use descriptive, lowercase names with hyphens
3. Update references in `index.html` if needed
4. Test the application to ensure images load correctly
